RetroPal

A fast, lightweight retro console emulator built for smooth classic gaming on Android phones, tablets, and TV. Responsive controls, optimized performance, and a clean interface for long play sessions.

Get on Play Store

Supported Systems

  • GB (Game Boy)
  • GBC (Game Boy Color)
  • GBA (Game Boy Advance)
  • NES (Nintendo Entertainment System)
  • SNES (Super Nintendo)

More systems may be added in future updates.

Features

  • Accurate retro console emulation
  • Smooth performance optimized for mobile and TV
  • External Bluetooth and USB controller support
  • Customizable on-screen controls
  • In-game save support (SRAM)
  • Fast-forward option
  • Clean, distraction-free interface
  • RetroAchievements integration (optional login)
  • Automatic cover art and metadata support
  • Lightweight and stable design

Important

RetroPal does not include any games. You must provide your own legally obtained game files (ROMs) to use this application.
